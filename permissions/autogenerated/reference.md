## Default Permission

Default permissions for the plugin

- `allow-is-exist`
- `allow-is-dir`
- `allow-is-file`
- `allow-size`
- `allow-name`
- `allow-extname`
- `allow-full-name`
- `allow-parent-name`
- `allow-icon`
- `allow-metadata`
- `allow-compress`
- `allow-decompress`
- `allow-transfer`
